# Another password manager
Учебный проект - программа для удобного хранения логинов и паролей от различных сервисов в зашифрованном виде.
## Основной функционал
- Поддержка нескольких пользоватей.
- Проверка сложности паролей.
- Возможность добавления сервисов (имя, адрес) и аккаунтов для них (логины, пароль, комментарий).
- Смена логотипа в окне авторизации на ПКМ. Двойной ЛКМ - сброс.
- Для хранения данных используется БД sqlite.
## Запуск программы (Windows, python 3.12)
### Ручная сборка:
Открыть консоль в папке, где будет папка с проектом. Создать виртуальное окружение:

    python -m venv folder_name
Активировать виртуальное окружение:

    cd folder_name
    Scripts\activate
Установить необходимые пакеты:

    pip install -r requirements.txt
Запустить саму программу:

    python main.py

### Скомпилированный файл:
Исполняемый файл программы находится в папке dist. В текущей директории он создает файл базы данных и файл с логотипом (если выбран).
Для перекомпиляции снова активировать виртуальное окружение и написать:

    pyinstaller --onefile --add-data "custom_widgets/common_password.txt:custom_widgets" your_script.py
